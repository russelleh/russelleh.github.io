Russell Emmer

I care about the tools people use to communicate, and I like using computers to build and play with those tools. You can see some of those things on this page.


Lojbo

I wanted to give others an easy way to understand the beauty of Lojban without requiring knowledge of linguistics or formal grammar. The page includes a searchable dictionary with features not found in other Lojban dictionaries.


Color

This applet uses results from the XKCD color survey to dynamically color the page background based on user input. Public survey data includes some rude words.


Doomsday

This binary clock counts down in Planck time to the Dark Era of the universe, when all black holes have evaporated. With five hundred and six bits, this clock finishes counting down approximately 10100 years from today.


Numerals

This tool parses numbers written in english, such as 'one' or 'eight hundred'. This page uses the PEG.js parsing library to compile javascript from a grammar file.


Base

This simple counter illustrates the relationship of different number bases by fixing text size to the logarithm of the base. I intended this project to convey the purpose of number base without the need for explanatory text.


Avalanche

The avalanche effect can be found in many cryptographic algorithms, particularly hash algorithms such as SHA. This page uses the checksum feature of credit card numbers to walk the user through a simple verification algorithm.


Endowment

Nick Bostrom's article entitled Astronomical Waste inspired me to create this. I like using time and big numbers to show simple things quickly.
